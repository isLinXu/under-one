| 对比项目 | 我们的差异 |

| **LangChain** | LangChain是"连接工具"，我们是"管理工具运行时"（调度+健康+降级） |
| **AutoGPT** | AutoGPT是"自主代理"，我们是"代理的运维基础设施"（8个运维技能） |
