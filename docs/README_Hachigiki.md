# 八奇技 · Agent 能力增强框架

> 基于《一人之下》八奇技设定，为 AI Agent 设计的九大能力增强技能体系

---

## 项目概述

八奇技是一套面向 AI Agent 的**工程化能力增强框架**。它不是聊天提示词模板，而是将 Agent 从"问答机器"升级为具备**自我诊断、工具调度、任务规划、知识管理**能力的自主执行体。

通过九大可独立部署、可协同调度的技能模块，Agent 能够在复杂长对话中保持稳态、在多工具场景中自动编排、在海量信息中提取洞察、在不确定环境下优雅降级。

## 技能详解

### 1. 炁体源流 · 稳态自愈引擎

```
核心机制:
  - 上下文熵计算: 冲突×2 + 缺口×1.5 + 冗余×0.5
  - DNA快照: O(1) 比对核心决策哈希，快速检测漂移
  - 三级蒸馏: 70%轻量压缩 → 85%精华提取 → 95%分段处理

触发条件（自动）:
  - 对话轮次 ≥ 10 轮
  - 检测到"不对/错了/矛盾/之前说"等关键词
  - 上下文使用率超过 80%

输出: JSON 健康报告 {entropy, consistency, alignment, health_score, alerts}
脚本: entropy_scanner.py
```

**实战示例**：你和 Agent 聊了 15 轮设计电商系统，第 12 轮确认用 MySQL，第 15 轮 Agent 突然回到 PostgreSQL。炁体源流在第 15 轮自动检测到 DNA 快照哈希不符（a3f7d2 ≠ b8e1c4），触发稳态修复，将对话拉回到最新确认版本，整个过程你几乎无感知。

---

### 2. 通天箓 · 瞬发指令工厂

```
核心机制:
  - 6大符箓类型: 分析/创作/验证/转换/检索/决策
  - 图论冲突检测: 拓扑排序 + 约束满足问题(CSP)
  - 禁咒分级: 🔴高危(删除/资金/医疗) → 强制确认

触发条件（自动）:
  - 用户说"帮我设计...流程/方案/系统"
  - 单次任务包含 ≥ 3 个能力维度

输出: JSON 符箓计划 {talisman_list, topology, execution_plan, curse_level}
脚本: fu_generator.py
```

**实战示例**：你说"看看这三家竞品上周做了什么，跟我们比怎么样，给老板写个摘要"。通天箓自动拆分为：①检索箓×3（并行爬网页）→ ②分析箓（提取功能点）→ ③决策箓（差距矩阵）→ ④创作箓（高管摘要）→ ⑤转换箓（Markdown组装）。全程无需你写提示词。

---

### 3. 大罗洞观 · 全局洞察雷达

```
核心机制:
  - 5种关联发现: 语义/时序/结构/指代/逻辑
  - 关联可信度: A级(>0.8) / B级(0.5-0.8) / C级(<0.5)
  - Mermaid 认知地图自动生成
  - 反向洞察: 从结论反推所需信息缺口

触发条件（自动）:
  - 输入文本 > 3000 字
  - 检测到跨段落指代词"这/那/上述"

输出: JSON 洞察报告 {links, knowledge_graph, mermaid_code, gaps}
脚本: link_detector.py
```

**实战示例**：Agent 分析一份 50 页的用户反馈报告，自动发现："所有'搜索不到文件'的抱怨，都和后台'索引同步延迟 2.3 小时'的监控日志在时间上重合"——这是你单独看任何一页都发现不了的跨段洞察。

---

### 4. 神机百炼 · 自主工具锻造

```
核心机制:
  - 器胚模板: argparse + logging + 异常处理 + dry-run
  - 进化之链: 器胚(v0.1) → 开锋(v0.5) → 试炼(v0.9) → 成品(v1.0)
  - 器灵契约: 输入/输出/性能/异常四维 API 规范
  - 自动测试生成: 正常/边界/异常 3类用例

触发条件（自动）:
  - 用户说"写个脚本/工具"
  - 同一操作在对话中出现 ≥ 3 次

输出: Python 脚本 + 测试脚本 + 契约文档
脚本: tool_factory.py
```

**实战示例**："帮我写个脚本，把文件夹里所有 JSON 的 email 字段校验一下，无效的单独列出来。"30 秒后你拿到一个带命令行参数、有异常处理、带测试用例的脚本，直接能跑。

---

### 5. 风后奇门 · 任务优先级引擎

```
核心机制:
  - 九维度评分: 紧急/重要/依赖/资源 + 天时/地利/人和
  - 八门映射: 开门(立即) → 生门(重点) → 景门(审视) → 杜门(延后) → 死门(终止)
  - 蒙特卡洛推演: 100 次随机扰动，评估计划鲁棒性
  - 后悔值分析: 执行后对比反事实方案

触发条件（自动）:
  - 检测到 ≥ 3 个待处理任务
  - 用户问"先做什么/怎么安排"

输出: JSON 排盘报告 {ranked_tasks, eight_gates, monte_carlo, execution_plan}
脚本: priority_engine.py
```

**实战示例**：周五下班前老板丢了 3 个任务。Agent 排盘后：修复生产 bug（开门，立即）→ 升级安全依赖（生门，重点）→ 写周报（死门，周一再说）。蒙特卡洛显示按时完成率 100%，无需额外缓冲。

---

### 6. 六库仙贼 · 知识消化器

```
核心机制:
  - 来源可信度: S(1.5×) / A(1.2×) / B(1.0×) / C(0.6×)
  - 消化率评估: 核心论点 + 支撑证据 + 应用场景
  - 保鲜期管理: 技术方案(3月) / 行业数据(6月) / 基础理论(3年+)
  - 艾宾浩斯反刍: 高消化间隔×1.5，低消化间隔×0.5

触发条件（自动）:
  - 输入包含 ≥ 3 个不同来源
  - 信息总量 > 2000 字

输出: JSON 消化报告 {digest_rate, knowledge_units, freshness_schedule}
脚本: knowledge_digest.py
```

**实战示例**：Agent 学习 RAG 技术，把 20 篇资料消化成 5 个知识节点，标注"这篇论文是核心（高消化，保鲜 3 年）""这篇知乎回答只是个人经验（低消化，3 个月后过期）"。

---

### 7. 双全手 · 人格与记忆守护者

```
核心机制:
  - 灵魂烙印 DNA: 价值内核/能力边界/诚信原则/尊重底线（不可变更）
  - 风格偏离度: |当前 - DNA期望| / DNA期望
  - 人格分裂防护: 单一会话角色锁定，3轮内切换>3次触发警报
  - 情绪时序分析: 追踪用户情绪变化，预判输出调整

触发条件（自动）:
  - 用户说"说人话/通俗点/正式点"
  - 跨会话首次对话
  - 连续 3 轮偏离度 > 0.3

输出: JSON 校验报告 {deviation_score, drift_level, dna_violations, can_switch}
脚本: dna_validator.py
```

**实战示例**：你说"用嘲讽的语气教我做投资"。双全手拦截：这会违背"尊重底线"和"能力边界（投资建议）"，拒绝切换。它可以保持轻松语气，但明确标注"投资建议请咨询专业人士"。

---

### 8. 拘灵遣将 · 多工具调度中枢

```
核心机制:
  - 灵体图鉴: 搜索/浏览/代码/数据/图像/语音/部署/诊断 8 类
  - SLA 契约: 预期耗时 + 硬超时 + 软超时 + 降级策略
  - 健康度监控: 优秀(>95%) / 良好(80-95%) / 警戒(60-80%) / 病态(<60)
  - 百鬼夜行: 大规模同质任务并行（分片→并行→聚合）

触发条件（自动）:
  - 任务需要 ≥ 2 个工具
  - 批量处理 ≥ 10 项相似子任务

输出: JSON 调度报告 {dispatch_plan, sla_log, fallback_count, health_report}
脚本: dispatcher.py
```

**实战示例**："分析三家竞品并生成图表"。Agent 自动调度搜索灵×3（并行爬网页）→ 代码灵（清洗数据）→ 绘图灵（生成柱状图）。如果某个网页打不开，自动跳过并标注数据来源缺失。

---

### 9. 八卦阵 · 中央协调器

```
核心机制:
  - 状态总线: 实时维护八技状态表（激活/休眠/故障）
  - 互斥仲裁: 3对互斥自动检测（符箓vs锻造/修复vs重排/洞察vs吸收）
  - 效能聚合: 全局平均分 + 健康度动态计算
  - 监控面板: ASCII 柱状图 + HTML 可视化

触发条件（自动）:
  - ≥ 2 个八技同时激活
  - 用户请求"查看八技状态/效能报告"

输出: JSON 协调报告 {status_bus, arbitration, aggregated_score, dashboard}
脚本: coordinator.py + dashboard.py
```

**实战示例**：复杂任务中，通天箓和神机百炼同时激活（互斥）。八卦阵仲裁：通天箓效能分 94.4 > 神机百炼 92.0，通天箓先执行，神机百炼排队，完成后自动唤醒。

## 快速开始

### 安装技能

将 `.skill` 文件放入 Agent 的技能目录：

```bash
# 用户技能目录（推荐）
cp *.skill /app/.user/skills/

# 或全局技能目录
cp *.skill /app/.agents/skills/
```

### 运行自检脚本

每个技能附带可执行 Python 脚本，可独立运行验证：

```bash
# 炁体源流 - 扫描上下文健康度
python qiti-yuanliu/scripts/entropy_scanner.py context.json

# 通天箓 - 生成符箓执行计划
python tongtian-lu/scripts/fu_generator.py "分析竞品并生成报告"

# 风后奇门 - 任务优先级排盘
python fenghou-qimen/scripts/priority_engine.py tasks.json

# 八卦阵 - 全局状态监控
python bagua-zhen/scripts/coordinator.py skills_status.json
```

## 技术亮点

| 维度 | 实现 |
|------|------|
| **自动化程度** | 9/9 技能具备自动触发器，无需用户显式调用 |
| **自检机制** | 每技能附带执行后效能闭环评分（1-100），驱动自我优化 |
| **失败回退** | 每技能定义 ≥ 3 种失败场景的自动降级策略 |
| **互斥协同** | 3对互斥技能 + 6条协同链路，八卦阵统一仲裁 |
| **可执行性** | 9个 Python 脚本可直接运行，产出结构化 JSON 报告 |
| **文化融合** | 以八奇技为隐喻，将抽象 Agent 能力具象为可理解的"术法" |

## 适用场景

- **长对话任务**：技术方案设计、产品需求分析、多轮决策讨论
- **多工具协同**：网页爬取 + 数据分析 + 可视化 + 报告生成
- **信息过载**：从数十篇文档/网页中提取核心洞察
- **任务优先级**：同时面对多个 deadline 时的全局规划
- **自动化工作流**：将重复性操作固化为可复用脚本
