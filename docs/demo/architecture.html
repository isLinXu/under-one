✨ 核心特性

🎯 零外部依赖

纯 Python 标准库实现，无需安装任何第三方包，即插即用。

🔧 配置外部化

所有阈值、权重支持从 YAML 配置文件读取，运行时动态调整。

🧠 自适应进化

修身炉引擎根据历史数据自动调整阈值，实现 Skill 自我优化。

🔗 跨 Skill 学习

KnowledgeHub 支持 Skill 间知识迁移，优化经验共享。

📊 实时监控

八卦阵生成 HTML 监控面板，实时展示十技健康状态。

🛡️ 降级保护

拘灵遣将提供工具不可用的降级策略，确保系统韧性。

🛠️ 技术栈

Python 3.8+ 标准库 only JSON YAML (可选) pytest GitHub Pages HTML5 CSS3 JavaScript
